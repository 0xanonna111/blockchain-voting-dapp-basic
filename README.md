# Blockchain Voting DApp Basic

This project helps beginners understand how blockchain can be used for fair and transparent voting.

You will learn:
- How a voting smart contract works
- How to connect a wallet
- How to read and write blockchain data
- How to show voting results on a website
- How DApps work in real life

## Requirements
- Node.js installed
- MetaMask browser extension
- Basic JavaScript knowledge

## Setup

1. Clone the repo
git clone https://github.com/yourusername/blockchain-voting-dapp-basic.git

2. Install packages
npm install

3. Run the app
npm start

4. Open browser
http://localhost:3000

## Project Structure

- index.html = Main webpage
- app.js = Frontend logic
- contract.js = Smart contract interface
- web3.js = Blockchain connection
- deploy.js = Contract deploy script
- config.js = Network settings
- server.js = Local server
- package.json = Project info
- utils.js = Helper functions
- .env.example = Environment example

## Features
- Connect wallet
- Show candidates
- Cast a vote
- Show live results
- Simple smart contract

## Learning Goal
This project shows how blockchain can be used to build trust in digital systems.
